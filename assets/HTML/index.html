Street Hockey

A modalidade é uma adaptação do Hockey Inline para atender as necessidades das pessoas que querem jogar, mas muitas vezes não tem as condições perfeitas para realizar o esporte segundo todas as regras oficiais.
Portanto é uma forma mais simples que as pessoas encontraram para jogar o Hockey.

Modalidades do Hockey

No Gelo: Em uma pista de bordas aredondadas, com patins de gelo.

Inline: Em quadras de concreto, com patins Inline.

Em Campo: Em campos gramados, sem patins e com o taco diferente.

Regras do Jogo

O Street Hockey foi pensado para facilitar a realização do esporte. Portanto pode ser realizado em qualquer superficie onde os patins tenham bom desenpenho. Também não há regras quanto ao tamanho do gol.

Jogadores

Não há um numero correto. Quanto maior o numero de jogadores, maiores as possibilidades estratégicas. Também não há necessidade de um goleiro.

Tempo

Deve ser combinado com os jogadores um limite específico de tempo, ou de pontos para que a partida acabe.

Equipamentos

Essenciais para a partida

Gols para marcar os pontos.

Essenciais para o jogador

Patins, capacete, tacos de Hockey.

Opcionais para o jogador

Caneleiras, joelheiras, cotoveleiras.

Campeonato aberto ao publico. Basta ter o patins e equipamentos de protecao essenciais. Os tacos sao disponibilizados.
